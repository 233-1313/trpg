属性调整值是指在进行检定时，基于属性获得的调整值。它们遵照下表进行换算：

|  属性值  | 调整值 |  属性值  | 调整值 |
| :---: | :-: | :---: | :-: |
|   1   | -5  | 16-17 | +3  |
|  2-3  | -4  | 18-19 | +4  |
|  4-5  | -3  | 20-21 | +5  |
|  6-7  | -2  | 22-23 | +6  |
|  8-9  | -1  | 24-25 | +7  |
| 10-11 |  0  | 26-27 | +8  |
| 12-13 | +1  | 28-29 | +9  |
| 14-15 | +2  |  30   | +10 |
